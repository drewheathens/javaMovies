# javaMovies.

## Description
javaMovies is a java application that:

 - Hits url.
 - Gets JSONArray.
 - Saves the data in 3rd normal form postgreSQL.

## Techonologies Used.

- JAVA 8

- postgreSQL

## Prerequisites

internet access to get the JSONArray.

## Installation 

- install postgresql 
- create database
-Run DB from file DB.java as main class.
